¿Que es HTML5?

Es un conjunto de 3 tecnologias

  1. Html
  2. Css
  3. JavaScript

Html

Html es un lenguaje de etiquetas que se podria decir así que es el esqueleto de la pagina web

Head

Es la cabeza del la pagina web sirve para importar archivos externos o librerias metas etc

Los meta no tienen su par de cierre al final se le incluye la barra de cierre / al final se recomienda seguir las reglas de sintaxis por compatibilidad

Body

Es el cuerpo de la pagina web todo lo que pongamos en el body se va a ver

Css

Es un lenguaje de Estilos si Html es el esqueleto Css son nuestras faciones externas

JavaScript

Este es un lenguaje de programacion que sirve para darle interaccion al usuario que no sean estaticas y para mas cosas
